## Context

`refresh-tui-chat-visual-language` 已经把排队消息从 timeline 中拆出，也把消息语言从日志风推进到了聊天风。但它把“主题存在感”推得过头了：整个 `Screen`、`#body`、`#timeline-scroll`、`#queue-tray` 和输入区都被显式背景色覆盖，结果是 TUI 不再像一个嵌入终端环境的应用，而更像一张整屏涂层。

这类问题不是简单审美偏好，而是终端 UI 的结构性约束：
- 终端默认底色本身就是应用背景；
- 边框与强调色应点到为止；
- 空态区域不该长期占空间；
- 布局必须能适应窄窗口，而不是依赖固定大块留白。

所以这次 change 的目标不是重做视觉，而是把已经走偏的视觉方案收敛回可持续的终端表面设计。

## Goals / Non-Goals

**Goals:**
- 恢复终端原生底板，让 SmartIPO 只在边框、焦点和轻强调上使用浅绿色。
- 让 queue tray 只在 `_pending_turns` 非空时显示，空态时完全隐藏。
- 减少固定高度、固定空白和大面积着色，提升不同终端宽高下的响应式表现。
- 保留已经正确的聊天语言方向，例如 `👨‍💻` / `🤖` 和更自然的正文表达。

**Non-Goals:**
- 不改变 FIFO 队列行为、turn 生命周期或旧事件隔离逻辑。
- 不重新引入 `你:`、`AI:` 或 timeline 内排队标签。
- 不新增复杂主题系统或多套皮肤切换。

## Decisions

### 1. 宿主终端默认背景优先，SmartIPO 主题色仅用于边框与强调

这次不再给 `Screen`、`#body` 等大容器显式设置背景色。终端原生底色继续承担全局背景职责，SmartIPO 只负责：
- 边框颜色；
- 焦点边框；
- 少量 panel / bubble 的强调色；
- 白色正文文字。

原因很简单：终端应用不是浏览器页面。把整块底色改掉会打破宿主环境的一致性，也会在不同主题的终端里显得突兀。

### 2. queue tray 采用按需挂载或按需隐藏，而不是空态占位

当 `_pending_turns` 为空时，queue tray 不显示，不返回“暂无排队消息”，也不保留固定最小高度。只有在确实有排队消息时才出现。

这是信息密度的基本纪律：没有信息就不要占位。与其给用户一个“空托盘”，不如把空间还给 timeline 和输入区。

### 3. 布局收缩优先于视觉铺满

所有辅助区域都按内容高度收缩，避免固定大块空白：
- status banner 保持紧凑；
- queue tray 只在有内容时出现；
- 输入区与 timeline 通过 margin/padding 维持节奏，而不是靠大面积背景块区分层级。

这样做不仅更优雅，也更有远见，因为它天然兼容窄终端、小窗口和侧边嵌入式使用场景。

### 4. 响应式重点放在“减少不可压缩元素”

终端环境没有浏览器那种丰富断点体系，所以最靠谱的响应式策略不是堆媒体查询，而是减少不可压缩元素：
- 去掉 queue tray 的固定 `min-height`；
- 减少不必要的双层 padding；
- 泡泡宽度按内容自适应，不做宽面板铺满；
- 让 timeline 始终保有最大的可用垂直空间。

### 5. 聊天视觉语言继续保留，但要避免再堆装饰

`👨‍💻` / `🤖`、自然正文和更轻的工具状态表达仍然保留。这里不回退，但也不再继续加重装饰层，比如额外空态文案、重色块或无必要的提示面板。

## Risks / Trade-offs

- [Risk] queue tray 隐藏后，用户可能一眼看不到“为什么托盘没了”。 → Mitigation：托盘只在有排队消息时出现，出现即代表状态变化，本身就是更清晰的语义。
- [Risk] 去掉大面积背景色后，界面层次感变弱。 → Mitigation：保留浅绿色边框、焦点和局部 bubble 强调，用留白而不是涂底做层次。
- [Trade-off] 不做复杂断点布局，意味着第一版响应式主要靠收缩而不是重排。 → 这是刻意选择；终端里“能缩、能藏、能留白”比“复杂重排”更稳。

## Migration Plan

1. 先收回全局背景色，只保留边框与强调色。
2. 再调整 queue tray 显隐逻辑，确保空态不挂载或不显示。
3. 最后收紧间距与固定高度，并更新测试验证文本区域边界。

## Open Questions

- queue tray 在多条消息排队时，是否需要设置一个轻量的最大显示高度，避免极端长队列时挤压 timeline。
- 后续是否需要把 status banner 也改成更轻量的单行摘要模式，以进一步压缩垂直空间。
