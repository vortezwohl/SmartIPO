## ADDED Requirements

### Requirement: 系统必须只清理已确认无语义作用的静态噪音
系统 MUST 只删除已经确认无语义作用的 import 和未使用形参。系统 MUST NOT 把包级重导出、抽象接口签名、测试替身签名或任何会改变调用约定的符号，当作普通 lint 噪音机械清理。

#### Scenario: 清理真正未使用的运行时形参
- **WHEN** 某个函数形参在函数体内未被读取，且该形参仅因统一 handler 签名而保留
- **THEN** 系统 MUST 通过前导下划线等保守方式显式标记其未使用状态
- **AND** 系统 MUST 保持该函数的调用签名和运行行为不变

#### Scenario: 识别不能删除的导入
- **WHEN** 某个 import 被用于 `__all__` 重导出、包入口暴露、抽象接口声明或测试替身契约
- **THEN** 系统 MUST 保留该 import
- **AND** 系统 MUST NOT 仅依据“本文件内未直接读取”就删除它

### Requirement: 系统必须为这类清理提供行为等价验证
系统 MUST 在完成静态清理后执行最小验证闭环，证明清理没有改变现有行为。

#### Scenario: 完成静态清理后的验证
- **WHEN** 系统完成本轮 import 和未使用形参清理
- **THEN** 系统 MUST 至少执行编译检查和现有测试
- **AND** 系统 MUST 确认清理前后运行时行为保持一致
